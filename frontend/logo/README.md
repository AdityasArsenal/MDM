# MDM Management System Logo

This folder contains the logo for the MDM (Mid-Day Meal) Management System.

## Logo File
- `mdm-logo.svg` - Main logo in SVG format

## Design Elements
The logo features:
- A blue circular background representing trust and reliability
- A white plate/bowl symbolizing food and meals
- Yellow circles representing rice/food items
- A spoon indicating meal service
- "MDM" text at the bottom

## Usage
The logo is used in:
- Dashboard header
- Application branding
- Documentation

## Colors
- Primary Blue: #3B82F6
- Dark Blue: #1E40AF
- White: #FFFFFF
- Yellow (Food): #FCD34D
- Gray (Spoon): #E5E7EB

The logo is also copied to `/public/mdm-logo.svg` for web access.
